Приложение 2
Порядок расчета страховых тарифов по Правилам № 18
добровольного страхования от болезней и несчастных случаев
на время поездки за границу
Страховой тариф по варианту равен:
произведению соответствующих базового страхового тарифа при страховании по варианту «A» и корректировочных коэффициентов, округленному в установленном порядке, – при страховании по варианту «A»;
произведению соответствующих базового страхового тарифа при страховании по варианту «B» и корректировочных коэффициентов, округленному в установленном порядке, – при страховании по варианту «B».
Страховой тариф по Застрахованному лицу равен:
страховому тарифу, рассчитанному в соответствии с подпунктом 1.1 пункта 1 настоящего Приложения, – при заключении договора страхования по варианту «A»;
сумме страховых тарифов, рассчитанных в соответствии с пунктом 1 настоящего Приложения,– при заключении договора страхования по варианту «A» и по варианту «B».
